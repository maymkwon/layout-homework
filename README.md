## 폴더 구조(client) <a name="structure"></a>


```
.src
+-- common
| +-- components : 공통 컴포넌트 폴더
| +-- layout : 레이아웃관련 스타일드 컴포넌트 폴더
| +-- utils : 기타 helper 함수 정의폴더
| +-- const.ts : 상수 파일
|
|+-- pages
| +-- config
|   +-- components : config 관련 컴포넌트
|   +-- index.ts : config 컨테이너
|
| +-- vm
|   +-- components : vm 관련 컴포넌트
|   +-- index.ts : vm 컨테이너


```
